# יומן מסחר אישי - בניית קובץ התקנה (.exe)

קובץ ה-ZIP הזה מכיל את כל קוד התוכנה. הוא בנוי כך שתוכל לקבל קובץ `.exe` אמיתי
להתקנה במחשב Windows, **בלי להריץ שום פקודה במחשב שלך** - הבנייה מתבצעת אוטומטית
בשירות חינמי של GitHub (נקרא GitHub Actions).

## הוראות - שלב אחר שלב

### שלב 1: יצירת Repository ב-GitHub

1. היכנס ל-[github.com](https://github.com) והתחבר לחשבון שלך
2. לחץ על הכפתור הירוק **"New"** (או על סימן ה-+ בפינה הימנית העליונה → **New repository**)
3. תן שם ל-Repository, לדוגמה: `trading-journal`
4. השאר אותו **Public** (כדי שה-Actions החינמיים יעבדו)
5. **אל תסמן** שום תיבה (לא README, לא .gitignore, לא License)
6. לחץ **Create repository**

### שלב 2: העלאת הקבצים

1. בעמוד שנפתח, תראה הוראות תחת הכותרת "...or push an existing repository"
2. **לחלופין, הדרך הפשוטה ביותר:** בעמוד ה-Repository הריק, לחץ על הקישור
   **"uploading an existing file"**
3. גרור לתוך החלון **את כל הקבצים והתיקיות** מתוך ה-ZIP שקיבלת (חלץ אותו קודם)
   - חשוב: ודא שהתיקייה `.github` (כולל הנקודה בהתחלה) מועלית גם היא -
     ב-Windows Explorer ייתכן שצריך להפעיל "הצג קבצים מוסתרים" כדי לראות אותה
4. למטה, כתוב הודעת commit כלשהי (לדוגמה: "התחלה") ולחץ **Commit changes**

### שלב 3: הפעלת הבנייה האוטומטית

ברגע שהקבצים הועלו, GitHub יתחיל אוטומטית לבנות את קובץ ה-exe (כי הגדרנו את זה
לקרות אוטומטית בכל העלאה). זה לוקח כ-10-15 דקות.

1. למעלה ב-Repository, לחץ על הלשונית **"Actions"**
2. תראה שם תהליך בנייה רץ (עיגול צהוב מסתובב). חכה שהוא יהפוך לסימן ✓ ירוק
3. לחץ על הבנייה שהושלמה

### שלב 4: הורדת קובץ ההתקנה

1. אחרי שהבנייה הסתיימה (✓ ירוק), לך ללשונית **"Releases"** בצד ימין של עמוד
   ה-Repository הראשי (או חפש "Releases" בתפריט הצדדי)
2. תראה שם גרסה חדשה שנוצרה אוטומטית, עם קובץ `.exe` מצורף תחת **"Assets"**
3. לחץ על קובץ ה-`.exe` כדי להוריד אותו

### שלב 5: התקנה

לחיצה כפולה על קובץ ה-`.exe` שהורדת - זה יתקין את האפליקציה כמו כל תוכנה רגילה,
עם אייקון בשולחן העבודה / בתפריט התחל.

**כל מי שתשלח לו את אותו קובץ `.exe`** יכול להתקין את זה אצלו באותו אופן - כל
אחד מקבל יומן ריק משלו, הנתונים לא משותפים בין משתמשים.

## פרטים טכניים (לידיעה)

- האפליקציה בנויה ב-**Tauri** (Rust + WebView מובנה של Windows) - לכן קובץ
  ההתקנה קטן יחסית (כמה MB, לא מאות MB כמו Electron)
- הנתונים נשמרים בקובץ JSON אמיתי בתיקיית הנתונים של המשתמש
  (`%APPDATA%\com.personal.tradingjournal\`), **לא** ב-localStorage של דפדפן -
  זה פותר לחלוטין את בעיית אובדן הנתונים שהייתה בגרסת ה-HTML.
